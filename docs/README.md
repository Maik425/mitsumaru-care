# みつまるケア プロジェクト設計書

## 📋 プロジェクト概要

**みつまるケア**は、介護施設の業務効率化を目的とした包括的なシフト・役割・勤怠管理システムです。

### 🎯 主要機能

1. **月間シフト作成** - 自動シフト生成と詳細調整
2. **役割表作成** - 業務担当者の自動割り当て
3. **勤怠管理** - 予定と実績の照合・管理

### 🔄 システム連携

- 各機能は独立して動作
- 登録情報を共有して連携
- 一貫したデータ管理で業務効率化
