# 業務フロー設計書

## 📋 1. 概要

このドキュメントでは、みつまるケアシステムの各機能における業務フローと処理の流れについて詳細に説明します。

### 1.1 設計方針

- **業務の自然な流れ**を反映したフロー設計
- **ユーザーの操作感**を重視した処理順序
- **エラーハンドリング**を含む包括的なフロー
- **データの整合性**を保つ処理の順序

### 1.2 フローの分類

1. **認証・認可フロー** - ログインから権限確認まで
2. **シフト管理フロー** - シフト作成から確定まで
3. **役割表管理フロー** - 役割割り当てから確定まで
4. **勤怠管理フロー** - 出退勤から承認まで
5. **マスターデータ管理フロー** - 各種設定データの管理

## 🔐 2. 認証・認可フロー

### 2.1 ログインフロー

```mermaid
flowchart TD
    A[ユーザーアクセス] --> B[ログイン画面表示]
    B --> C[メールアドレス・パスワード入力]
    C --> D{入力値検証}
    D -->|NG| E[エラーメッセージ表示]
    E --> C
    D -->|OK| F[Supabase認証]
    F --> G{認証結果}
    G -->|失敗| H[認証エラー表示]
    H --> C
    G -->|成功| I[JWTトークン取得]
    I --> J[ユーザー情報取得]
    J --> K[権限確認]
    K --> L{権限レベル}
    L -->|OWNER/ADMIN| M[管理者ダッシュボード]
    L -->|MEMBER| N[一般職ダッシュボード]
```

### 2.2 権限チェックフロー

```mermaid
flowchart TD
    A[API呼び出し] --> B[JWTトークン検証]
    B --> C{トークン有効性}
    C -->|無効| D[401 Unauthorized]
    C -->|有効| E[ユーザー情報取得]
    E --> F[テナント情報取得]
    F --> G[権限レベル確認]
    G --> H{必要な権限}
    H -->|不足| I[403 Forbidden]
    H -->|充足| J[処理実行]
```

## 📅 3. シフト管理フロー

### 3.1 月間シフト作成フロー

```mermaid
flowchart TD
    A[管理者アクセス] --> B[シフト作成画面表示]
    B --> C[対象月選択]
    C --> D[シフト形態確認]
    D --> E[職員情報確認]
    E --> F[配置ルール確認]
    F --> G[休暇情報確認]
    G --> H[自動シフト生成]
    H --> I{生成結果}
    I -->|エラー| J[エラー表示・修正]
    J --> G
    I -->|成功| K[シフト詳細表示]
    K --> L[手動調整]
    L --> M[シフト確定]
    M --> N[職員への通知]
```

### 3.2 シフト詳細設定フロー

```mermaid
flowchart TD
    A[シフト一覧表示] --> B[対象シフト選択]
    B --> C[シフト詳細画面]
    C --> D[職員割り当て確認]
    D --> E{調整が必要}
    E -->|はい| F[職員変更]
    F --> G[役職変更]
    G --> H[時間調整]
    H --> I[保存]
    E -->|いいえ| J[確定]
    I --> K[変更内容確認]
    K --> L{承認}
    L -->|承認| M[シフト確定]
    L -->|差戻し| N[修正依頼]
    N --> F
```

### 3.3 休暇管理フロー

```mermaid
flowchart TD
    A[職員アクセス] --> B[休暇申請画面]
    B --> C[休暇種別選択]
    C --> D[日付選択]
    D --> E[理由入力]
    E --> F[申請送信]
    F --> G[管理者通知]
    G --> H[管理者確認]
    H --> I{承認判断}
    I -->|承認| J[休暇確定]
    I -->|却下| K[却下理由入力]
    K --> L[職員通知]
    J --> M[シフト調整]
    L --> N[申請修正]
    N --> F
```

    F --> G
    G --> H[業務実行]
    H --> I{退勤時刻}
    I -->|予定時刻前| J[早退申請]
